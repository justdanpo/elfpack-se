BookManager v2.8


Умеет показывать/закрывать/переключать книги/жаву.
Умеет показывать/закрывать/переключать/показывать_копирайты эльфов.
В первой вкладке на цифровых клавишах - быстрый запуск для штатных менюх.
Во второй - быстрый запуск эльфов.
"#" - свернуть всё.
долгий "#" (в первой вкладке) - закрыть всё.
Возможность назначать ярлыки из самого эльфа.


необходимые файлы необходимо положить в $DIR_INI$/bookman/:

bookman.ini - названия книг для отображения на первой вкладке
shortcuts.ini - значения кнопок для быстрого запуска (создается самим эльфом)
файлы картинок для закладок (если необходимо)


отображение имён (bookman.ini):

  что_видим : что_хотим_видеть

  Standby: Режим ожидания
  Desktop: Главное меню

описание ярлыков (shortcuts.ini):

[S_KEY0] - быстрый запуск по короткому нажатию "0" в первой вкладке.
[L_KEY0] - быстрый запуск по долгому нажатию "0" в первой вкладке.
[ES_KEY0] - быстрый запуск эльфа по короткому нажатию "0" во второй вкладке.
[EL_KEY0] - быстрый запуск эльфа по долгому нажатию "0" во второй вкладке.


Запуск java-приложения:

Пишем по маске: java:Name//Vendor
Name - Название приложения
Vendor - Поставщик приложения
Оба параметра можно посмотреть в "Информации" у приложения

[S_KEY0]: java:Jimm//Jimm Project
 

Описание полей в bookman.bcfg:

"Start Button" - кнопка запуска BookManager'а
"Tabs images" - имена файлов с картинками для вкладок
"Ignore Keylock" - Разрешить/запретить вызов менеджера при заблокированной клавиатуре
"Display elfs on 1 tab" - Отображать/не отображать на первой вкладке эльфы с гуи-интерфейсом
"Close when inactive" - Разрешить/запретить закрытие менеджера при неактивности пользователя
"Name Books" - Называть книги как сессии, к которой они относятся, или брать название самих книг
"Show after shortcut" - поднимать StandbyBook вверх списка после вызова быстрой ссылки. Для владельцев w800 обязательно должно стоять Current book, чтобы работали ярлыки
"Active Tab" - номер вкладки которую необходимо показывать при запуске BookManager'а
